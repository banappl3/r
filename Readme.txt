Das in diesem Archiv enthaltene Listing GrafikModell2.R erzeugt eine Grafik für die mittleren Monatstemperaturen in Basel zwischen 2006 und 2013. Gezeigt werden die gemessenen Daten,
der Trend und das saisonale Modell. Das Code-Fragment setzt voraus, dass Sie vorab alle im Heftartikel gedruckten Zeilen für dieses Beispiel in R eingegeben haben. Laden Sie anschließend den Code in R über die Eingabe von:

source("GrafikModell2.R")

Über die im Artikel gezeigte Ausgabe hinaus wird die Grafik zusätzlich als PDF gespeichert. 

Falls Sie die im Heftartikel gedruckten R-Eingaben nicht einzeln nachvollzogen haben, können Sie statt dessen das Listing Basel.R laden:

source("Basel.R")

und anschließend

source("GrafikModell2.R")

In jedem Fall ist aber zuvor nötig, die Textdatei mit den Wetterdaten für Basel wie im Heft beschrieben von der Webseite von MeteoSchweiz herunterzuladen.
